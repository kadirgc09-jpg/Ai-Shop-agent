# Haustier-Shop Chatbot START

Das ist dein neuer verkaufbarer Chatbot-Prototyp fuer Haustier-Shops.

## Ziel

Du willst kleinen Shopify-Shops einen Chatbot verkaufen.

Dieser Bot ist fuer Shops mit Haustier-Produkten gedacht, zum Beispiel:

- Hundefutter
- Katzenfutter
- Snacks
- Spielzeug
- Pflegeprodukte
- Zubehoer

## Was der Bot beantworten kann

Der Bot kann typische Kundenfragen beantworten:

1. Welches Futter passt zu meinem Tier?
2. Wie lange dauert der Versand?
3. Kann ich Ware zurueckgeben?
4. Gibt es Futter fuer sensible Hunde oder Katzen?
5. Welche Produkte sind fuer Welpen oder Kitten geeignet?
6. Was mache ich bei Allergien?
7. Wie erreiche ich den Support?

## Welche Datei du in n8n importierst

Importiere diese Datei in n8n:

```text
workflows/n8n-pet-shop-chatbot.json
```

## Ganz einfach gesagt

Dieser Workflow macht:

```text
Kundenfrage rein -> Bot versteht die Frage -> passende Antwort raus
```

## Wichtig

Das ist Version 1.

Version 1 ist ohne OpenAI API gebaut, damit du erstmal testen kannst.

Version 2 bauen wir danach mit OpenAI API, damit der Bot viel besser und natuerlicher antwortet.

## Verkaufsidee

Du kannst Kunden sagen:

```text
Ich baue Ihnen einen KI-Chatbot fuer Ihren Haustier-Shop.
Er beantwortet Kundenfragen zu Futter, Versand, Rueckgabe und Produkten.
Dadurch sparen Sie Zeit und Kunden bekommen schneller Hilfe.
```
