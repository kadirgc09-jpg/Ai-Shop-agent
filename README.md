# Ai-Shop-agent

Das ist dein Projekt fuer einen **n8n Chatbot fuer Haustier-Shops**.

## Wichtig: Du brauchst nur diese eine Datei

```text
workflows/n8n-pet-shop-chatbot.json
```

Diese Datei importierst du in n8n.

## Ganz einfache Schritte

1. Oeffne die Datei `workflows/n8n-pet-shop-chatbot.json`
2. Kopiere den kompletten Inhalt
3. Gehe in n8n
4. Klicke auf Import
5. Fuege den kopierten JSON-Text ein
6. Importieren
7. Workflow starten

## Was der Bot macht

Der Bot beantwortet einfache Kundenfragen fuer Haustier-Shops, zum Beispiel:

- Futter
- sensibler Magen
- Allergien
- Versand
- Rueckgabe
- Welpen und Kitten
- Zubehoer
- Support

## Was aktuell NICHT drin ist

Noch keine echte OpenAI-KI.

Das ist Version 1 zum Testen. Danach bauen wir Version 2 mit OpenAI API, damit der Bot viel besser und natuerlicher antwortet.

## Dein Ziel

Du willst so einen Bot spaeter an Shopify-Shops verkaufen.

Ein einfacher Verkaufs-Satz waere:

```text
Ich baue Ihnen einen KI-Chatbot fuer Ihren Haustier-Shop,
der Kundenfragen zu Futter, Versand, Rueckgabe und Produkten beantwortet.
```
