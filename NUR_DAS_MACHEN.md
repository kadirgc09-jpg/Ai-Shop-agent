## Was du als Naechstes machst

### Schritt 1

Oeffne dein GitHub-Projekt.

### Schritt 2

Klicke auf diese Datei:

```text
data/lead_input.csv
```

### Schritt 3

Dort steht gerade ein Beispiel:

```csv
name,website,niche
Demo Shop,https://example.com,Mode
```

### Schritt 4

Spaeter ersetzt du das mit echten Shops, zum Beispiel:

```csv
name,website,niche
Shop 1,https://example-shop.de,Mode
Shop 2,https://anderer-shop.de,Kosmetik
```

## Ganz einfach gesagt

Du gibst Shop-Links rein.

Der Agent macht daraus eine vorbereitete Nachricht.
